Dynamic Programming Fibonacci Sequence Calculator

Environment:
- Developed using C programming language.
- No additional libraries are required.

How to Compile and Run:
1. Ensure you have a C compiler installed (e.g., GCC).
2. Save the  source code in a file named "fibonacci.c".
3. Open a terminal or command prompt.
4. Navigate to the directory where the "fibonacci.c" file is located.
5. Compile the code by entering the following command:
 gcc -o fibonacci fibonaccic
6. Run the compiled executable:
   ./fibonacci

7. The program will output the results for addition, multiplication, subtraction, and the Fibonacci sequence.
